# 🚀 Portfolio Website - Abyan Maheswara

Welcome to the official repository of my portfolio website. This is a modern, interactive, and high-performance web profile built to showcase my projects in **IoT (Internet of Things)**, **AI Engineering**, and **Industrial Automation**.

🔗 **Live Demo:** [abyanmaheswara.github.io/abyan-maheswara/](https://abyanmaheswara.github.io/abyan-maheswara/)

---

## ✨ Key Features

- **🤖 Premium AI Chatbot Assistant**: A custom-built, simulated AI assistant that can answer questions about my background, projects, and skills. Includes typing animations and smart responses.
- **🎨 Glassmorphism & Modern UI**: Sleek design with blurred backgrounds, smooth gradients, and a semi-transparent aesthetic.
- **🌗 OS-Synced Dark/Light Mode**: Automatically detects and synchronizes with your device's system appearance (Dark/Light mode) in real-time.
- **📊 Google Analytics Integration**: Full visitor tracking and engagement analytics implemented via G-Tag.
- **⚡ Performance Optimized**:
  - **Native Lazy Loading**: All assets are only loaded when needed.
  - **Smooth Animations**: Powered by AOS (Animate On Scroll) and GSAP.
  - **3D Particles**: Dynamic background powered by Three.js.
- **📱 Fully Responsive**: Optimized for every screen size, from desktops to mobile devices.
- **🖼️ Interactive Gallery**: Features a custom modal zoom for personal and project photos.

---

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
- **Animations**: [GSAP](https://greensock.com/gsap/), [AOS](https://michalsnik.github.io/aos/)
- **3D Graphics**: [Three.js](https://threejs.org/)
- **Analytics**: Google Analytics 4
- **Icons**: [Simple Icons](https://simpleicons.org/)
- **Deployment**: GitHub Pages

```bash

cd abyan-maheswara
# Open index.html in your favorite browser
```
